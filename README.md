# 蓝博航发研学基地网站

基于 Next.js App Router 与 Tailwind CSS 搭建的单页官网项目。

## 启动方式

1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

3. 打开浏览器访问

`http://localhost:3000`

## 项目结构

- `app/page.jsx`：主页内容
- `app/layout.jsx`：全局布局与站点 metadata
- `app/globals.css`：Tailwind 与全局样式
- `tailwind.config.js`：Tailwind 配置
